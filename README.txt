
Para uma melhor performance da aplicação e localização de ficheiros:


1º Abrir a pasta "vbnet-avaliacao\resources\database"
	Executar no SQL Server o ficheiro "DataBase20180803.sql"

2º Abrir Visual Studio

3º Efectuar a paradidade com as portas
	Abrir pasta "\vbnet-avaliacao\resources\SetupVSPE"
	Instalar o ficheiro "SetupVSPE.msi"
	A porta escolhida para a aplicação é a 3

4º Arrancar com o arduino no proteus
	Abrir a pasta do projecto "vbnet-avaliacao\resources"
	Importar a código do Arduino ficheiro: "SR232_teste.ino.hex"

5º Abrir a pagina de default no web browser pelo visual net

6º Efectuar o start da aplicação(clicar na aplicacao com o botao direito do rato e em seguida  em 'set as start project' se necessario)
